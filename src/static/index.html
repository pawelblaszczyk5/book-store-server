Available endpoints

/

Current endpoint

GET /getBook?id=bookId

Get single book by ID

GET /getLimitedReviews?id=bookId&limit=number&skip=number

Get reviews for given book ID, limit and skip by given number

GET /getNumberOfReviews?id=bookId

Get reviews number for given book ID

GET /getRecommendedBooks

Get random books

GET /getBestsellers

Get random bestsellers

GET /getNew

Get random new books

GET /getSales

Get random on sale books

POST /getBooks

Get books based on filter, example filter structure:

    searchPhrase: string;
    dateFrom: number;
    dateTo: number;
    genres: Array<string>;
    sortBy: string;
    sale: boolean;
    bestseller: boolean;
    new: boolean;

POST /sendContact

Send contact message

POST /register

Register user

POST /login

Login user

POST /authenticate

Check user credentials

POST /userData

Get user data after authentication

POST /getBooksByIds

Get books by array of IDs

POST /addReview

Add review if authenticated and bought book

POST /placeOrder

Place order
